# 🌿 Magic Plants Diary

A simple web app to track plants, care routines and growth over time.

Built as a personal project to learn frontend development and database integration, evolving into a real tool for everyday use.


## ✨ Features

- 🌱 Create and manage plant profiles
- 💧 Track watering, treatments and notes
- 📒 Event-based plant diary
- 🔍 Dedicated plant pages with history
- ⚡ Instant event insertion
- 🧠 Structured for future smart suggestions


## 🧱 Tech Stack

- HTML, CSS, JavaScript
- Supabase (PostgreSQL database)
- Git & GitHub for versioning


## 🚀 Current Status

Core features are implemented and stable:

- Plant creation ✔
- Event tracking ✔
- Database integration ✔
- UUID migration completed ✔
- Shopping list ✔
- Soil ingredients tracking ✔
- Plant editing ✔


## 🛣️ Next Steps

- 🧠 Smart care suggestions (watering reminders, alerts)
- 👤 User authentication (multi-user support)


## 💡 Project Vision

The goal is to evolve this app from a simple tracker into a smart assistant for plant care, accessible and usable by anyone — even non-technical users.
